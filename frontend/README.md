# Dependency-smells-detector frontend

React Web Application Layer for visualising backend analysis JSON artefacts.

## Features

- Loads the `.json` output generated by the backend.
- Renders the dependency graph with Cytoscape.
- Colors nodes by highest smell severity:
  - Critical: red
  - High: orange
  - Medium: yellow
  - Low: blue
  - No smells: green
- Filters the graph by smell type and severity.
- Shows node metadata, SSSS score, evidence, and detector source.

## Commands

 - To install dependencies, run:
```powershell
    npm install
```

 - To run the development server, run:
 ```powershell
    npm run dev
 ```

  - To build the production bundle, run:
  ```powershell
      npm run build
  ```
